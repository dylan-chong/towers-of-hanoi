# Assignment

Run tests:

    ./gradlew clean test

and view test results in `build/reports/tests/test/index.html`

Run program from Gradle:

    ./gradlew clean run

Run program with jar:
    
    java -jar submit/app.jar # sequential
    java -jar submit/app.jar --parallel

Code is in `src/main/`

Tests are in `src/test/`
